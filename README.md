# OTP2Week3ClassAssignment

This project is a JavaFX application that allows users to input their first name, last name, and email, and save this information to a MySQL database. The application supports multiple languages, including English, Japanese, and Russian.

## Project Structure

- `src/main/java/org/otp2/otp2week3classassignment/HelloApplication.java`: The main entry point of the application.
- `src/main/java/org/otp2/otp2week3classassignment/HelloController.java`: The controller class that handles user interactions.
- `src/main/resources/hello-view.fxml`: The FXML file defining the user interface.
- `src/main/resources/bundle_EN.properties`: Resource bundle for English language.
- `src/main/resources/bundle_JP.properties`: Resource bundle for Japanese language.
- `src/main/resources/bundle_RU.properties`: Resource bundle for Russian language.

## Dependencies

The project uses the following dependencies:

- JavaFX for the user interface.
- MySQL Connector/J for database connectivity.
- JUnit for testing.
